# Contributing to Sui Documentation Portfolio

This is a personal portfolio project, but feedback is welcome!

## How to Provide Feedback

1. **Open an issue** - For suggestions or corrections
2. **Discuss** - Use GitHub Discussions for questions
3. **Contact** - Reach out on [LinkedIn](https://linkedin.com/in/luisjguedes)

## Reporting Errors

If you find technical errors:
1. Specify the document and section
2. Explain the issue
3. Suggest a correction (if possible)

## Style Feedback

Suggestions for improving clarity or structure are appreciated.

Thank you for your interest!
